Claude overview

Think with Claude. Build with Claude Code.

Claude is Anthropic’s AI assistant for understanding, reasoning, writing, and creating. Claude Code is the software-building side of that relationship: it can inspect a project, edit files, run commands, test the result, and work with you through every decision.
