01The story of the fortnight

Meet Claude Sonnet 5.

If you only read one update, read this one — it's the model now doing the work behind Claude Code for most people.

Jun 30, 2026New model

A faster, cheaper model that acts on its own.

Sonnet 5 is Anthropic's new mid-tier model built for work: it can make plans, use tools like browsers and terminals, and run through multi-step jobs on its own, checking itself as it goes. Anthropic says it closes much of the gap with the flagship Opus 4.8 while costing a fraction as much. It's the new default for Free and Pro users, and it became the default model inside Claude Code too.

Jul 7–11Claude Code

Claude Code: five releases in five days

A steady run of updates (versions 2.1.203 → 2.1.207) sharpened the agent workflows and fixed a lot of rough edges: a full /doctor setup checkup, protection against tampering with session transcripts, smarter linking of sessions to their pull requests, big Windows reliability fixes, a clearer agents view with live status, and auto-update downloads that stream to disk to save roughly 400 MB of memory.

Claude Code changelog →
Jul 9, 2026Feature
Worth a look

Reflect: a mirror for how you use Claude

A new dashboard (Settings → Reflect) shows the topics you spend time on, your most active day and peak hour, and gentle observations about how you work with Claude. It's in beta on Free, Pro, and Max — on the web and desktop — and needs memory switched on. Sensitive subjects are shown only at a high level, health topics are excluded, and Anthropic says the insights aren't used for anything else.

Anthropic newsroom →
Jul 9, 2026Company

Ben Bernanke joins Anthropic's trust

Former Federal Reserve chair Ben Bernanke was appointed to Anthropic's Long-Term Benefit Trust, the governance body designed to keep the company accountable to its mission as it grows.

Anthropic newsroom →
Jul 8, 2026Editors

VS Code adds multi-chat for Claude sessions

Visual Studio Code 1.128 lets a single Claude agent session hold several related chats, so you can compare approaches side by side, branch from an earlier point in the conversation, and run work in parallel — a small change that makes exploring options far less fiddly.

Visual Studio Magazine →
Jul 7, 2026Product
Worth a look

Cowork breaks free of the desktop

Claude Cowork — the assistant that works alongside you across your files and apps — arrived on web and mobile, starting with Max subscribers. Sessions and files now follow you between devices, tasks keep running in the background even with no device online, and you can approve steps from your phone. Anthropic notes that more than 90% of Cowork use isn't coding at all — it's everyday business and content work.

Claude blog →
Jul 7, 2026Connectors

Claude can now write to Microsoft 365

The Microsoft 365 graduated from read-only to hands-on: with an administrator's consent, Claude can draft and send email, manage calendar events, tidy mailbox settings, and create or update files in OneDrive and SharePoint. (The Teams connector stays read-only for now.)

Claude release notes →
Jul 6, 2026Feature story

The making of Claude Code

Anthropic told the origin story of its coding agent — how it grew from a one-person internal experiment into a product used across the whole company, with engineers shipping dozens of internal releases a day. The team even debated whether to release it publicly at all. One striking number: pull-request throughput rose about 67% as the engineering team doubled.

Anthropic features →
Jul 2, 2026Safety

More on Fable 5's safeguards

Following Fable 5's global redeployment on July 1, Anthropic published more detail on the model's cyber safeguards and proposed an industry framework for grading how severe a jailbreak really is — an attempt to make "the model was jailbroken" a measurable claim rather than a headline.

Anthropic newsroom →
Jun 30, 2026Product

Claude Science: a workbench for researchers

Anthropic launched Claude Science, an environment that brings research tools and flexible computing together in one place for scientists — aimed at the kind of literature-heavy, data-heavy work that doesn't fit a normal chat window.

California partners with Anthropic

In a first-of-its-kind agreement, California state agencies can access Claude at a roughly 50% per-seat discount, bundled with free workforce training and hands-on technical help — a sign of how quickly these tools are moving into government work.

The model line-up, in plain terms.

Claude isn't one model — it's a small family, each tuned for a different job. In Claude Code you can switch between them any time with /model. Tap a name to copy its API id.
